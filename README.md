# Student Management REST API

## Description
This project is a REST API built using Node.js HTTP module without external frameworks.

## Features
- Create Student (POST)
- Get All Students (GET)
- Get Student by ID (GET)
- Update Student (PUT)
- Delete Student (DELETE)

## Technologies Used
- Node.js (http module)

## How to Run
1. Install Node.js
2. Run:
   node server.js

## API Endpoints

POST /students  
GET /students  
GET /students/:id  
PUT /students/:id  
DELETE /students/:id
